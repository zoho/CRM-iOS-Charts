## Installation

ZohoCRMCharts is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'ZohoCRMCharts', :git => 'https://github.com/zoho/CRM-iOS-Charts.git
```
